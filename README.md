# Personal Bucket List

A personal bucket list webpage created using HTML and CSS to list and track life goals and experiences.

## Technologies Used
- HTML
- CSS

## Features
- Clean and simple webpage layout
- Organized list of personal goals
- Styled with CSS for a pleasant look

## How to Run
1. Download or clone the repository
2. Open `index.html` in any browser

- CSS styling and layout
